# Pixel Rogue Cards

一款像素风格的卡牌构筑类肉鸽游戏 (Roguelike Deckbuilder)

## 🎮 游戏特色

- 🃏 **卡牌构筑**：收集卡牌，构建你的专属卡组
- 🏛️ **遗物系统**：获得强大遗物，改变游戏策略
- 🗺️ **分支路线**：选择你的冒险路径
- 👹 **多样敌人**：每种敌人都有独特机制
- 💀 **永久死亡**：失败后重新开始，但每次都会变强
- 🎨 **像素风格**：复古像素美术风格

## 🎯 如何游玩

### 在线试玩
直接访问：https://D-L123456.github.io/pixel-rogue-cards

### 本地运行
1. 克隆仓库
```bash
git clone https://github.com/D-L123456/pixel-rogue-cards.git
cd pixel-rogue-cards
```

2. 用浏览器打开 `index.html`

或使用本地服务器：
```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 🕹️ 操作说明

- **鼠标左键**：选择卡牌、点击按钮、选择路线
- **数字键 1-5**：快速打出对应卡牌
- **空格键**：结束回合
- **R键**：重新开始（游戏结束后）

## 🎴 游戏机制

### 卡牌类型
- **攻击卡**：造成伤害
- **防御卡**：获得护甲
- **技能卡**：特殊效果（抽牌、增益等）
- **能力卡**：持续整场战斗的被动效果

### 遗物类型
- **战斗遗物**：增强战斗能力
- **经济遗物**：获得更多金币
- **恢复遗物**：战斗后恢复生命
- **特殊遗物**：改变游戏规则

### 路线选择
每层结束后选择：
- ⚔️ **战斗**：普通敌人，获得卡牌/金币
- 👹 **精英**：强力敌人，获得稀有遗物
- 🏪 **商店**：购买卡牌和遗物
- ❓ **事件**：随机事件，可能获得奖励或惩罚
- 💤 **休息**：恢复生命值
